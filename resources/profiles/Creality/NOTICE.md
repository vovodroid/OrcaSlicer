# Creality vendor profile attribution

## K-series filament profiles imported from CrealityPrint

The 230 filament presets in this directory whose name matches the pattern
`* @Creality K2 [Plus|Pro]? <nozzle> nozzle.json` were imported verbatim from
CrealityOfficial/CrealityPrint v7.1.1 (`resources/profiles/Creality/filament/`).

- Source: https://github.com/CrealityOfficial/CrealityPrint
- License: GNU Affero General Public License v3.0 (compatible with OrcaSlicer's
  AGPL-3.0 — see top-level LICENSE.txt)
- Imported: 2026-05-19

Brand coverage includes Hyper PLA / PETG / ABS / PA-CF / PLA-CF, CR-PLA /
CR-PETG / CR-ABS / CR-Silk / CR-TPU and the corresponding Generic / eSUN /
PolyMaker / Panchroma variants Creality ships profiles for.

These presets reference machine names already present in OrcaSlicer's
existing Creality vendor (e.g. `Creality K2 0.4 nozzle`) so no machine
profile changes were required.
